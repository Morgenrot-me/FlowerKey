# 浏览器插件目录说明

## 目录作用

FlowerKey Chrome/Edge 浏览器插件（Manifest V3），提供悬浮球快速密码生成和完整管理侧边栏。

## 文件列表

| 文件/目录 | 作用 |
|-----------|------|
| `manifest.json` | 插件清单（参考用，实际构建使用 public/manifest.json） |
| `public/manifest.json` | 构建源清单，Vite 直接复制到 dist/ |
| `public/icons/` | 插件图标（16/48/128px） |
| `vite.config.ts` | Vite 多入口构建配置 |
| `background/` | Service Worker 后台脚本 |
| `content/` | 注入网页的 Content Script |
| `popup/` | 工具栏弹出窗口（备用） |
| `sidepanel/` | 侧边栏完整管理界面 |
| `dist/` | 构建产物（gitignore，加载到浏览器的目录） |

## 技术说明

- Manifest V3，使用 `sidePanel` 权限
- 点击工具栏图标自动打开侧边栏（`setPanelBehavior`）
- 构建命令：`pnpm build` / 开发监听：`pnpm dev`
