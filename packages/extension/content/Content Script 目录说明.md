# Content Script 目录说明

## 目录作用

注入到所有网页的内容脚本，提供悬浮球 + 快速密码计算浮层。

## 文件列表

| 文件 | 作用 |
|------|------|
| `index.ts` | 完整的 content script：悬浮球、浮层面板、拖拽、钉住、密码生成、自动填充 |

## 技术说明

- 使用 Shadow DOM（closed 模式）隔离样式，避免与页面 CSS 冲突
- 悬浮球支持自由拖拽，松手自动吸附左/右边缘
- 面板支持 header 拖动自由定位，关闭时重置位置
- 钉住（📌）状态：点击外部不关闭面板，面板显示浅蓝边框
- ⚙ 按钮展开/收起字符集和长度配置行（默认隐藏）
- 密码生成防抖 500ms，输入时自动触发
- 移动端（宽度 ≤ 600px）面板居中显示、宽度自适应

## 注意事项

- 不能直接调用 `chrome.sidePanel.open()`，管理面板只能通过点击工具栏图标打开
