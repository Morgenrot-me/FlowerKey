# 浏览器插件目录说明

## 目录作用

FlowerKey 的 Chrome/Edge 浏览器插件入口，基于 Manifest V3 WebExtension 标准。

## 文件列表

### 根目录

| 文件 | 作用 |
|------|------|
| `manifest.json` | WebExtension Manifest V3 配置 |
| `vite.config.ts` | Vite 构建配置，多入口打包 |
| `package.json` | 包配置和依赖 |

### popup/

| 文件 | 作用 |
|------|------|
| `index.html` | Popup HTML 入口 |
| `main.ts` | Popup Vue 应用挂载 |
| `App.vue` | Popup 主组件 - 快速密码生成界面 |

### sidepanel/

| 文件 | 作用 |
|------|------|
| `index.html` | SidePanel HTML 入口 |
| `main.ts` | SidePanel Vue 应用挂载 |
| `App.vue` | SidePanel 主组件 - 完整管理界面 |

### background/

| 文件 | 作用 |
|------|------|
| `service-worker.ts` | Background Service Worker - 右键菜单、消息通信 |

### content/

| 文件 | 作用 |
|------|------|
| `index.ts` | Content Script - 表单检测、密码填充、书签保存 |

## 技术说明

- Manifest V3，兼容 Chrome 和 Edge
- Vite 多入口构建（popup/sidepanel/background/content）
- 通过 `chrome.runtime.sendMessage` 进行组件间通信

## 扩展方式

- 新增页面：在对应目录创建 HTML + Vue 入口
- 新增权限：修改 `manifest.json` 的 permissions

## 注意事项

- Background 为 Service Worker，无 DOM 访问
- Content Script 运行在网页上下文，注意安全隔离
