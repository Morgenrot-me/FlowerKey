# 侧边栏目录说明

## 目录作用

Chrome 侧边栏（Side Panel）完整管理界面，点击工具栏图标打开。

## 文件列表

| 文件 | 作用 |
|------|------|
| `index.html` | 侧边栏入口 HTML |
| `main.ts` | Vue 应用挂载入口 |
| `App.vue` | 侧边栏根组件，加载 UI 组件库的完整管理视图 |

## 技术说明

- 通过 `chrome.sidePanel.setPanelBehavior({ openPanelOnActionClick: true })` 自动绑定工具栏图标
- 与 content script 通过 `chrome.runtime.sendMessage` 通信
- 解锁状态从 `chrome.storage.session` 读取
